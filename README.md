# COVID-19 Country Analysis

Data analysis notebook exploring COVID-19 statistics across countries using `covid19countryinfo.csv` dataset. 

## Key Visualizations

- **Top 10 Countries by Total Cases** (Bar chart)
- **Total Cases vs Total Deaths** (Scatter plot - country-wise aggregation)
- **Cases by First Case Date** (Line graph - sorted by date)

## Technologies Used
- **Pandas**
- **Matplotlib**
- **Numpy**
